# Superstore Sales Analysis - SQL

## Overview
SQL analysis of 9,000+ retail orders using SQLite.

## Business Questions Answered
1. Total Sales & Profit
2. Best Region by Sales
3. Most Profitable Category
4. Sub-Categories with Losses
5. Discount Impact on Profit (CASE)
6. Top 5 Customers
7. Top Sales Month
8. Customers Above Average Sales (Subquery)
9. Sales by Region (JOIN)
10. Top Customers by Sales & Profit (UNION)
11. Regions Above Average (Subquery in HAVING)
12. Regions Above Average (Subquery in FROM)

## Tools
- SQL / SQLite
- DB Browser for SQLite

## Key Insights
- Tables sub-category generates losses of $64K
- High discounts (>20%) cause negative profit
- Central Asia has highest average sales per order
